Personal CV Website – Farah Munawar

This project is a clean, responsive personal CV website built using HTML, CSS, and JavaScript.  
It is designed to present academic background, skills, and experience in a professional format.


Features:

- Light / Dark Mode Toggle
- Downloadable CV (Print → Save as PDF)
- Subtle animations for better user experience**
- Contact form (email delivery via Formspree)


Technologies Used:

- HTML5  
- CSS3  
- JavaScript
